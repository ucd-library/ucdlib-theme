Prebuilt Theme Package

The library offers a prebuilt bundle that can load the UC Davis brand theme into your webpage using a single loader script. This script will load the brands global css, icons, fonts and web components.

Import Theme

                
                  <script src="https://brand.library.ucdavis.edu/ui/loader.js"></script>

Flags

The loader.js endpoint accepts optional ?flags= parameter. You can provide multiple flags, comma separated.
Example: ?flags=ignoreFonts,ignoreIcons

  • ignoreCss: Do not load the global UCD Theme css
  • ignoreFonts: Do not load the Proxima Nova font set
  • ignoreIcons: Do not load the UCD Theme icon sets
  • ignorePolyfills: The loader.js script will detect if your browser does not support ES Classes. If your browser does not support ES Classes, by default, it will load a set of polyfills from Babel. However, if you wish to skip this step, provide the ignorePolyfills flag. All modern browsers support ES Classes, so this is an edge use case.

Web Component Polyfills

The loader.js endpoint will detect if your browser supports web components. If not, a set of polyfills will be loaded. All modern browsers support web components, this is an edge use case that any polyfills will be loaded.
